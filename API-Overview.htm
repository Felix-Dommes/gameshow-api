GET /joinPlayer?name=<player name>

400 Bad Request, if name is empty
OK Returns Text: <actual set player name>

GET /getPlayerData

OK Returns JSON: [{name: "", money: 0, money_bet: 0, vs_player: "", answer: 0}]

GET /betMoney?name=<name>&money_bet=<money>

406 Not Acceptable, if currently not in betting state for betting question
400 Bad Request, if name was not found or money_bet is invalid
OK Returns nothing

GET /attackPlayer?name=<name>&vs_player=<attacked player>

406 Not Acceptable, if currently not in selecting state for versus question
400 Bad Request, if name or vs_player was not found or they are equal
OK Returns nothing

GET /answerQuestion?name=<name>&answer=<ID>

406 Not Acceptable, if currently not in answering state
400 Bad Request, if name was not found or answer is invalid
OK Returns nothing

GET /getJokerFiftyFifty?name=<name>

406 Not Acceptable, if currently not in answering state for normal or betting question or all jokers were already used
400 Bad Request, if name was not found
OK Returns JSON [1, 2] (IDs of wrong answers)

GET /getGameEvents

OK Returns JSON: [{id: 0, event_name: "<event name>", event: {"<event name>": {<event related attributes>}}}]
event_names: BeginNormalQAnswering, BeginBettingQBetting, BeginBettingQAnswering, BeginEstimationQAnswering, BeginVersusQSelecting, BeginVersusQAnswering, ShowResults, GameEnding

POST /giveMoney

JSON Parameter: {name: "", money: 0}
name = player name, money = money to give
400 Bad Request, if name was not found
OK Returns JSON: {name: "", money: 0}
name = player name, money = total money after operation

POST /setJokers

JSON Parameter: {name: "", jokers: 0}
name = player name, jokers = number of available jokers
400 Bad Request, if name was not found
OK Returns JSON: {name: "", jokers: 0}
name = player name, jokers = number of available jokers

GET /kickPlayer?name=<player name>

400 Bad Request, if name was not found
OK Returns nothing

GET /activateNextQuestion

406 Not Acceptable, if currently not in results state
OK Returns nothing

GET /forceQuestionAnswering

406 Not Acceptable, if currently not in betting question betting or versus question selecting state
OK Returns nothing

GET /forceQuestionResults

406 Not Acceptable, if currently not in answering state
OK Returns nothing

GET /setNextQuestion?number=<question ID>

406 Not Acceptable, if currently not in results or game ending state
400 Bad Request, if number does represent a valid question number (must be 1 - len(questions))
OK Returns previously set question ID

POST /loadQuestions

JSON Parameter: {filename: ""}
filename = JSON file name inside the questions folder
406 Not Acceptable, if currently not in results or game ending state
400 Bad Request, if questions could not be loaded (e.g. file not found or wrong format)
OK Returns Text: <number of available questions>
